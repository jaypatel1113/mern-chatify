# CHATIFY


#### Created using :
_MongoDB_
_NodeJS_
_ExpressJS_
_ReactJS (Material UI)_
_SocketIO_

## Functionality
- Authentication
- One on One chat
- Group Chat
- Typing Indicator
- Status Indicator (Send, Deliver or Seen)
- Real Time Messaging
- Real Time Notification 
- Search other User
- Add and Remove from Group (Only Group Admin)
