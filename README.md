# CHATIFY


#### Created using :
_MongoDB_
_NodeJS_
_ExpressJS_
_ReactJS (Material UI)_
_SocketIO_

## Functionality
- Authentication
- One on One chat
- Group Chat
- **End to End Encryption**
- **Typing Indicator**
- **Status Indicator (Send, Deliver or Seen)**
- **Real Time Notification**
- Real Time Messaging
- Search other User
- Add and Remove from Group (Only Group Admin)
